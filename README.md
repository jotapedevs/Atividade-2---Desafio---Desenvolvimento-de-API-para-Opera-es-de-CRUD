# Atividade-2---Desafio---Desenvolvimento-de-API-para-Opera-es-de-CRUD

### Rotas

- **GET `/users`**: Retorna todos os usu치rios salvos no banco.
- **POST `/users`**: Cria um novo usu치rio.
- **PATCH `/users/:id`**: Atualiza um usu치rio existente pelo `id`.
- **DELETE `/users/:id`**: Deleta um usu치rio existente pelo `id`.

### Crie um .env
   ```env
   PORT=8000
   MONGODB_URL=URL do MongoDB, ex: mongodb+srv://#####.#####.mongodb.net
   MONGODB_USERNAME=SeuLoginNoMongoDB
   MONGODB_PASSWORD=SuaSenhaNoMongoDB
   ```
